Pigment.O

Operators

Letter : Middle Value of the Channel. Except in Hue case where it selects nearest Pure Hue.
Minus One : Minus one point on the Channel.
Slider : Move Slider to select color. Press any modifier key to pin to nearest percetual value or purest Hue.
Plus One : Plus one point on the Channel.
Value : Numerical value of the Channel.
Hex : Hex color code of the Selected Color.

Options

RGB : Activate RGB color space panel. RGB = Red Green Blue
HSV : Activate HSV color space panel. HSV = Hue Saturation Value. Works in RGB color space.
TTS : Activate TTS color mixer panel. TTS = Tint Tone Shade. "Select" color with double click and "Clear" color with any modifier key and double click.
MIX1 : Activate MIX1 color mixer panel. Select costum colors to mix in RGB interpolation. "Select" color with double click and "Clear" color with any modifier key and double click. Press any modifier key to pin to nearest percetual value.
MIX2 : Activate MIX2 color mixer panel. Select costum colors to mix in HSV interpolation. "Select" color with double click and "Clear" color with any modifier key and double click. Press any modifier key to pin to nearest percetual value.
PANEL : Pick HSV Sat and Value with display (HSV) or without any color display (NIL). "Highlight" selected color with Right Mouse Button or with Pen vertical Tilt (Up or Down).
