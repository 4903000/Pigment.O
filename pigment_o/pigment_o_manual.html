Operators

RGB Tab : RGB Color Space Picker.
R : Red Channel.
G : Green Channel.
B : Blue Channel.
HSL Tab : HSL Color Space Picker.
H : Hue Channel.
S : Saturation Channel.
L : Light Channel.
LAB Tab : LAB Color Space Picker.
L : Light Channel.
A : A Channel.
B : B Channel.
HEX Tab : HEX Color Code.
Code Input : Hue Channel.
C : Copy from Clipboard.
P : Paste from Clipboard.
