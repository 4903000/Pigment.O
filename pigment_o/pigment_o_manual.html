Pigment.O

Brush

Tip : Hidden Button on the Top Left. "Save" current BrushSize, Opacity and Flow, by pressing (Shift or Ctrl or Alt) and double Click. Double click to "Apply" previously saved settings.

Operators

Letter : Middle Value of the Channel. Except in Hue case where it selects nearest Pure Hue.
Minus One : Minus one point on the Channel.
Slider : Move Slider to select color. Press any modifier key to pin to nearest percetual value or purest hue.
Plus One : Plus one point on the Channel.
Value : Numerical value of the Channel.
Hex : Hex color code of the Selected Color.

Options

A : Activate Alpha / BW color space panel. Alpha=Transparency BW = Black White
RGB : Activate RGB color space panel. RGB = Red Green Blue
HSV : Activate HSV color space panel. HSV = Hue Saturation Value. Works in RGB color space.
HSL : Activate HSL color space panel. HSV = Hue Saturation Lightness. Works in RGB color space.
CMYK : Activate HSV color space panel. CMYK = Cyan Magenta Yellow Key.

COR : Activate COR color palette with 10 slots. Double click to "Apply" color. Shift or Ctrl and double click to "Save" color. Alt and double click to "Clean" color.
TTS : Activate TTS color mixer panel. TTS = Tint Tone Shade. "Select" color with double click and "Clear" color with any modifier key and double click. Press any modifier key to pin to nearest percetual value.
MIX : "Select" color with double click and "Clear" color with any modifier key and double click. Press any modifier key to pin to nearest percetual value.
MIX RGB : Select costum colors to mix in RGB interpolation.
MIX HSV : Select costum colors to mix in HSV interpolation.
MIX HSL : Select costum colors to mix in HSL interpolation.
MIX CMYK : Select costum colors to mix in CMYK interpolation.

PANEL : Left Mouse Button to "Select" color in panel plane. "Highlight" selected color with Right Mouse Button or with Pen vertical Tilt (Up or Down).
PANEL HSV : Pick HSV Sat and Value with display.
PANEL HSV0 : Pick HSV Sat and Value without any color display.
PANEL HSL : Pick HSL Sat and Lightness with display.
PANEL HSL0 : Pick HSL Sat and Lightness without any color display.

Updates

Time Check : Every second there will be a Krita color Check (default). This delta time can be changed to be slower (more than 1000) or disabled (less than 1000) on the code(check_timer). 1000ms = 1second.
Mouse Enter : Entering the Pigment.O panel will update itself and stops the timer.
Mouse Leave : Leaving the Pigment.O panel starts the timer.
